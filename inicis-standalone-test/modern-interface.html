INICIS 현대적 인터페이스 테스트

⚠️ 중요

BILLAUTH를 사용하면 구형 인증서 인터페이스가 나타납니다.

아래 방법들은 현대적 인터페이스를 유지하면서 정기결제를 구현하는 방법입니다.

방법 1: 일반결제로 카드 등록

일반결제를 0원 또는 100원으로 진행하고, 결제 성공 후 카드 정보를 저장

acceptmethod: "HPP(1):below1000:va_receipt"

방법 2: 정기결제 표시 + 일반결제

offerPeriod를 사용해 정기결제임을 표시하고 일반결제 진행

offerPeriod: "M2" (월 정기결제)

방법 3: 카드 자동 저장 옵션

일반결제 + 카드 저장 동의 옵션

acceptmethod: "HPP(1):below1000:cardautostore"

방법 4: 간편결제 정기결제

카카오페이, 네이버페이 등 간편결제의 정기결제 기능 사용
